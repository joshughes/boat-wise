1. Install TideWise from the HACS custom repository.
2. Confirm the installed TideWise version.
3. Add TideWise from the dashboard card picker.
4. Open the visual editor.
5. Select the Cherry Grove, SC station.
6. Save the card.
7. Refresh the dashboard.
8. Confirm the card still loads.
9. Change fishing mode.
10. Save and refresh again.
11. Toggle fishing score off/on.
12. Test with missing optional weather/surf entities.
13. Test on desktop.
14. Test on phone.
15. Screenshot any layout issue.
16. Copy browser console errors.
17. Report Home Assistant version.
18. Report HACS version.
19. Report browser/device.
20. Report whether installed from release, HACS custom repository, manual resource, or main branch.
21. If successful, star the repo or open a Works For Me / Confirmed Station report.
